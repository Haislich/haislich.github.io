# Machine Learning
## Theory
- [x] [Data science foundamentals, IBM, IBM cognitive class](https://cognitiveclass.ai/learn/data-science) : This course is a very basic approach to data science. The path per se will be barely an introduction, but if you don't even know where to start is a good place, plus at the end there's a free Linkedin shareable certificate.
- [x] [Applied Data science with Python, IBM, IBM cognitive class](https://cognitiveclass.ai/learn/data-science-with-python): This is a direct application of the concepts learnt through the previous course. It introduces the most popular Python libraries and explain how to start using them, as before it guarantees a free shareable Linkedin certificate.
